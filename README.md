# User Information Management System
A simple CRUD application using jQuery, AJAX, PHP, and MySQL.

### Features
- Real-time form validation (JS/jQuery).
- Image uploading using AJAX.
- Dynamic data grid (No page refresh).
- Edit/Modify existing records.

### How to Run
1. Import `database.sql` into your local MySQL (XAMPP).
2. Place files in `htdocs`.
3. Open `localhost/phpmyadmin/index.php`.
